-Introduction:

A basketball information website on Heroku that included the following features:

1. A ranking webapp that used the PageRank algorithm to rank the basketball teams based on their game results in a time period specified by the user.

2. A social news webapp that allowed users to read, submit and vote for news about basketball; the news items were ranked based on both the number of votes and time of submission.


-For local testing:

$ cd rankProject
$ source venv/bin/activate
$ python manage.py runserver
